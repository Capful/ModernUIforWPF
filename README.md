Modern UI for WPF （中文优化）
==============
```
A fork from Modern UI for WPF on Codeplex.com(https://mui.codeplex.com/)


Changes:

-Changed default target platform to .NET 4.0

-Changed default font and font size for optimizing Chinese characters displaying

-Removed FirstFloor.ModernUI.WPF4 project

-Removed [Build] folder

Please notice:
In the latest version of official ModernUI for WPF, dependence of 'Microsoft.Windows.Shell.dll' was removed when targeting to .NET 4.5.
I haven't merge this feature yet because this fork is based on .NET 4.0.
```

```
移植自Codeplex.com上的Modern UI for WPF项目(https://mui.codeplex.com/)

改动:

-默认的目标平台更改为.NET 4.0

-更改了一些默认字体值、字体大小，显示中文时变得更清晰

-移除原项目中的WPF4项目

-移除原项目中的Build目录


注意：
在新版的ModernUI for WPF中，.NET 4.5项目移除了对Microsoft.Windows.Shell.dll的依赖，但由于此fork基于.NET 4.0，所以并没有合并此特性
```
